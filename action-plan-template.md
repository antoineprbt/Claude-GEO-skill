# 90-Day GEO Action Plan Template

Customize this template based on audit findings. Prioritize actions by impact and effort.

## Month 1: Technical Foundation & Quick Wins (Days 1-30)

### Week 1: AI Crawler Access (Day 1-7)
- [ ] Audit and update robots.txt to allow all AI crawlers
- [ ] Verify the site is indexed in Bing Webmaster Tools
- [ ] Submit sitemap to Bing Webmaster Tools
- [ ] Check JavaScript rendering — if CSR, plan SSR migration or pre-rendering
- [ ] Create and deploy llms.txt file

### Week 2: Schema Foundation (Day 8-14)
- [ ] Add Organization schema to every page (header/footer)
- [ ] Add Article + Person (author) schema to all blog/content pages
- [ ] Add BreadcrumbList schema site-wide
- [ ] Validate all schema with Google Rich Results Test

### Week 3: Content Quick Wins (Day 15-21)
- [ ] Audit H1/H2 hierarchy on top 10 traffic pages
- [ ] Rewrite vague H2s to be descriptive, question-format
- [ ] Add answer islands (40-80 word direct answers) after each H2 on top 5 pages
- [ ] Add "Last updated: [date]" visible timestamps to key pages
- [ ] Update `dateModified` in schema markup

### Week 4: Meta & Technical Polish (Day 22-30)
- [ ] Audit and optimize title tags for top 20 pages
- [ ] Write compelling meta descriptions for top 20 pages
- [ ] Add alt text to all images on key pages
- [ ] Fix any HTTPS mixed content issues
- [ ] Run PageSpeed Insights and fix critical performance issues

**Month 1 KPI:** AI Citability Score improvement of +15-25 points

## Month 2: Content Optimization & Expansion (Days 31-60)

### Week 5-6: Deep Content Restructuring (Day 31-44)
- [ ] Identify 10 most important pages for AI visibility
- [ ] Restructure each page using the TLDR-first pattern
- [ ] Add FAQ sections (with FAQPage schema) to 5 key pages
- [ ] Add statistics and source citations to all key pages (target: 1 stat per 200 words)
- [ ] Add expert quotes or attribution where relevant
- [ ] Create comparison tables for product/service pages
- [ ] Add HowTo schema to any step-by-step content

### Week 7-8: Content Gap Filling (Day 45-60)
- [ ] Identify the top 20 questions users ask AI engines about your niche
- [ ] Create or update content to directly answer each question
- [ ] Build 2-3 pillar pages for core topics (3000+ words, comprehensive)
- [ ] Create 5-10 supporting cluster pages linked to pillar pages
- [ ] Ensure all new content follows GEO best practices (answer islands, factual density, schema)

**Month 2 KPI:** 20+ pages fully GEO-optimized, FAQ sections on 5+ key pages

## Month 3: Authority Building & Multi-Platform (Days 61-90)

### Week 9-10: Multi-Platform Presence (Day 61-74)
- [ ] Create or optimize YouTube presence (at least 3-5 videos covering core topics)
- [ ] Identify and engage in 5 relevant Reddit communities (authentic participation, not spam)
- [ ] Optimize LinkedIn company page with complete, keyword-rich description
- [ ] Create/claim profiles on G2, Capterra, or Trustpilot (if B2B/SaaS)
- [ ] Verify Wikipedia page exists or create Wikidata entry (if brand qualifies)
- [ ] Ensure entity consistency: brand name, description, facts identical across all platforms

### Week 11-12: Digital PR & Brand Mentions (Day 75-90)
- [ ] Identify 10 journalists/bloggers in your niche
- [ ] Pitch 3-5 story angles or data-driven insights for coverage
- [ ] Guest post on 2-3 industry publications
- [ ] Create a original research report or data analysis that others will cite
- [ ] Monitor brand mentions across AI engines (manual testing or tool)

**Month 3 KPI:** Presence on 8+ AI-cited platforms, 2+ press mentions

## Ongoing (Post 90 Days)

### Monthly Maintenance
- [ ] Update key pages with fresh data and current-year statistics
- [ ] Test AI visibility: run 20 core queries monthly on ChatGPT + Perplexity
- [ ] Track citation trends: are you being cited more or less over time?
- [ ] Monitor competitors' AI visibility
- [ ] Publish 4-8 new GEO-optimized content pieces per month
- [ ] Engage weekly on Reddit and YouTube

### Quarterly
- [ ] Full GEO re-audit
- [ ] Update schema markup for new content
- [ ] Review and update robots.txt
- [ ] Refresh pillar pages with latest data
- [ ] Evaluate new AI platforms entering the market

## Priority Matrix

Use this to customize the plan based on audit findings:

| Audit Score | Focus | Rationale |
|-------------|-------|-----------|
| 0-39 (Invisible) | 90% Month 1, 10% Month 2 | Must fix technical barriers before anything else |
| 40-59 (At Risk) | 50% Month 1, 40% Month 2, 10% Month 3 | Technical mostly OK, content needs heavy work |
| 60-79 (Needs Work) | 20% Month 1, 40% Month 2, 40% Month 3 | Foundation exists, optimize content + build authority |
| 80+ (AI-Ready) | 10% Month 1, 30% Month 2, 60% Month 3 | Focus on authority, PR, and expanding presence |
